Welcome to the Raiden Wizard

The Raiden Wizard is a tool to help you in your first steps to get started with running an operating a Raiden Node. By following through with the wizard, we will be able to help you:

  1. Generate an Ethereum account (a keystore file along with its secret passphrase) that should be used exclusively by Raiden
  2. Generate the configuration file that will be used by Raiden
  3. Get instructions on how to fund this account so that Raiden can be operational
  4. Download and install the latest version of Raiden
  5. Launch Raiden with the existing configuration
